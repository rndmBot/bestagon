Acknowledgments

This area is wide and deep, and it would be impossible without community support.

On this topic I want to give credit to those whose work helped me a lot on this quest.
I have learned a lot, studying all these frameworks and I give my big thanks to all these people:

1. `John Bywater <https://www.linkedin.com/in/johnbywater/>`__ and his project `Event Sourcing in Python <https://eventsourcing.readthedocs.io/en/stable/>`__.

2. `Axon Framework <https://www.axoniq.io/framework>`__ - ES framework for Java, their documentation is a great source of information about ES and DDD.

3. `Alexey Zimarev <https://www.linkedin.com/in/alexeyzimarev/>`__ and his project `Eventuous <https://eventuous.dev/go/intro/>`__.
